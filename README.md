# Grønn byguide Stavanger

**Her er guiden 👉 https://haomys.github.io/fivh-stavanger/**

## 🌍Guiden viser deg aktører og aktiviteter som fremmer bærekraftige praksiser og tenkemåter 🌍
Guiden er en nettapplikasjon som gir deg oversikt over steder i Stavanger og omegn hvor du kan reparere, kjøpe brukt og bærekraftig, låne, leie mm.
